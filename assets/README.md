# 静态资源文件

请将以下图片文件放入此目录：

## 必需文件

1. **icon.png** - 应用图标 (1024x1024 px)
2. **splash.png** - 启动屏图片 (1242x2436 px)
3. **adaptive-icon.png** - Android 自适应图标 (1024x1024 px)
4. **favicon.png** - Web 图标 (16x16 或 32x32 px)

## 可选文件

- 可以添加单词相关的图片到 images/ 子目录
- 可以添加自定义字体到 fonts/ 子目录

## 快速开始

如果没有图片资源，可以使用 Expo 的默认资源，或创建简单的彩色背景图作为占位符。

### 使用 Expo 默认图标
运行以下命令生成默认图标：
```bash
npx expo prebuild
```
