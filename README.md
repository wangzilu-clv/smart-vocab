# SmartVocab 3.0 - 智能背单词 APP

一款基于 Expo (React Native) 开发的智能背单词应用，通过智能推荐算法和错词本功能帮助用户更高效地学习英语单词。

## ✨ 核心功能

### 1. 学习功能
- 📚 **单词卡片** - 美观的卡片展示，包含音标、释义、例句
- 🎯 **学习模式** - 仅卡片模式，专注学习新词
- 🔄 **立即复习** - 学习完成后单词立即加入待复习列表
- 📝 **错词本** - 自动收集模糊/不认识/答错的单词

### 2. 复习功能
- **两轮复习机制**
  - 第一轮：**选择题模式** - 4选1选择正确释义
  - 第二轮：**拼写模式** - 看中文释义输入英文单词
- ⏰ **智能复习间隔** - 5分钟、30分钟、2小时、6小时、1天...
- 📊 **复习优先级** - 掌握度低的单词优先复习

### 3. 错词本（3.0新功能）
- 📝 **自动收集** - 学习时选择"模糊/不认识"，复习时答错自动加入
- 📖 **查看管理** - 专门的错词本页面查看所有错词
- 🔄 **自动复习** - 每天首次打开APP，错词自动加入当日复习列表
- 🗑️ **移除功能** - 掌握后可从错词本移除

### 4. 智能推荐算法
- **词根词缀关联** - 学过 "unhappy"，自动推荐 "unfair", "unknown"
- **主题关联** - 学过 "apple", "banana"，推荐其他水果类单词
- **难度递进** - 从简单到复杂逐步推荐
- **使用场景** - 根据偏好推荐（商务、旅游、学术、科技）
- **随机打乱** - 避免总是推荐a开头的单词

### 5. 数据与统计
- 🔥 **打卡记录** - 连续学习天数统计
- 📊 **学习统计** - 学习进度可视化
- 🏆 **成就系统** - 学习里程碑奖励
- 📖 **词库管理** - 按分类浏览所有单词

## 🚀 运行方法

### 前置要求
- Node.js 16+ 
- npm 或 yarn
- Expo CLI (`npm install -g expo-cli`)
- Expo Go App (iOS/Android)

### 安装依赖

```bash
cd smart-vocab
npm install
```

### 启动开发服务器

```bash
npm start
# 或
expo start
```

### 构建 APK

```bash
# 使用 EAS 构建
npm install -g eas-cli
eas login
eas build --platform android --profile preview
```

## 🔬 智能推荐算法详解

推荐算法综合考虑以下因素：

1. **词根词缀关联 (30%)**
   - 分析已学单词的词根、前缀、后缀
   - 推荐具有相同词根词缀的新单词

2. **主题关联 (25%)**
   - 根据已学单词的主题
   - 推荐同一主题下的相关单词

3. **难度递进 (20%)**
   - 新手优先推荐简单单词
   - 随着学习进度逐步增加难度

4. **场景偏好 (15%)**
   - 根据用户设置的使用场景偏好
   - 优先推荐相关分类单词

5. **多样性 (10%)**
   - 随机打乱推荐顺序
   - 避免总是推荐a开头的单词
